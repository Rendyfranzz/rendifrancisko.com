## Description

Proud to share my first project using flutter with the team, we successfully developed an innovative application for PT Petrokimia Gresik. This application streamlines the process of tracking goods in the port, enhancing operational efficiency.
Port-Pass is an automation system used to track loading-unloading of goods and inventory at PT Petrokimia Tbk.


## Project Overview

**Port Pass App** is an innovative solution developed to streamline the tracking of goods in the port, specifically for PT Petrokimia Gresik. This application is designed to enhance operational efficiency by automating the process of loading and unloading inventory.

## Key Features

- **Real-Time Tracking**: Monitor the status of goods in real-time as they are loaded and unloaded.
- **Inventory Management**: Keep track of inventory levels and movements within the port.
- **User-Friendly Interface**: An intuitive interface designed for ease of use by port staff.
- **Automated Reporting**: Generate reports automatically to keep stakeholders informed.

## Challenges Faced

- **Integration with Existing Systems**: Ensured seamless integration with PT Petrokimia Gresik's existing systems.
- **Data Accuracy**: Implemented robust validation to maintain accurate records of goods movements.
- **Scalability**: Designed the system to handle increased traffic and scale as needed.

## Technology Stack

- **Flutter**: Used for building a cross-platform mobile application that runs smoothly on both iOS and Android devices.
- **Firebase**: Integrated for real-time database and authentication services.
- **Dart**: Programming language used in conjunction with Flutter for app development.
